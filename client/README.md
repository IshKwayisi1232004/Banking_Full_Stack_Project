# Client (Frontend)

React + TypeScript frontend for the banking project.

## Run locally
```bash
cd client
npm install
npm run dev
```

Default app URL: `http://localhost:5173`

## Build
```bash
cd client
npm run build
```

## Main folders
- `src/pages` - route pages (`LoginPage`, `HomePage`, `TransactionPage`)
- `src/components` - reusable UI components
- `src/services` - API calls to backend (`http://localhost:5001`)
- `src/types` - shared TypeScript types
