# Server Source (`server/src`)

This folder contains all backend application code.

## Structure
- `server.ts` - server bootstrap
- `app.ts` - express app setup and middleware
- `config/` - environment config helpers
- `db/` - DB pools and DB utilities
- `modules/auth/` - authentication and JWT
- `modules/accounts/` - account APIs
- `modules/transactions/` - transaction coordinator and failpoints
- `utils/` - shared helpers
- `types/` - local TypeScript declarations
